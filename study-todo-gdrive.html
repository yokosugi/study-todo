⚙️ GOOGLE DRIVE 連携設定
初回のみ設定が必要です。以下の手順でAPIキーを取得してください。
1
Google Cloud Consoleでプロジェクトを作成
https://console.cloud.google.com/ を開く
→ 上部の「プロジェクトを選択」→「新しいプロジェクト」
→ プロジェクト名は何でもOK（例: study-todo）→「作成」
2
Google Drive API を有効化
「APIとサービス」→「ライブラリ」
→ Google Drive API を検索 →「有効にする」
3
OAuth同意画面を設定
「Google Auth Platform」→「設定」→「開始」
→ アプリ名・メールを入力、対象：外部 で完了
4
OAuthクライアントIDを作成
「クライアント」→「クライアントを作成」→ 種類：ウェブアプリケーション
→「承認済みのJavaScript生成元」に追加：
  • （このページのURLを貼り付け）
→「作成」→ クライアントID をコピー
5
テストユーザーを追加
「対象」→「テストユーザー」→「ユーザーを追加」
→ 自分のGmailアドレスを追加 →「保存」
クライアントID を貼り付け
💡 クライアントIDはこのデバイスのブラウザにのみ保存されます。
