⚙️ GOOGLE DRIVE 連携設定
初回のみ設定が必要です。以下の手順でAPIキーを取得してください。
1
Google Cloud Consoleでプロジェクトを作成
https://console.cloud.google.com/ を開く
→ 上部の「プロジェクトを選択」→「新しいプロジェクト」
→ プロジェクト名は何でもOK（例: study-todo）→「作成」
2
Google Drive API を有効化
左メニュー「APIとサービス」→「ライブラリ」
→ 検索欄に Google Drive API と入力
→ 「Google Drive API」を選択 → 「有効にする」
3
OAuth同意画面を設定
「APIとサービス」→「OAuth同意画面」
→ ユーザーの種類：外部 を選択 → 「作成」
→ アプリ名（例: 学習Todo）、メールアドレスを入力
→ 「保存して次へ」を3回クリックして完了
4
OAuthクライアントIDを作成
「APIとサービス」→「認証情報」→「認証情報を作成」→「OAuthクライアントID」
→ アプリの種類：ウェブアプリケーション
→ 「承認済みのJavaScript生成元」に以下を追加：
  • （このページのURLを貼り付けてください）
→ 「作成」→ 表示された「クライアントID」をコピー
5
テストユーザーを追加（外部アプリの場合）
「OAuth同意画面」→「テストユーザー」→「ユーザーを追加」
→ ご自身のGmailアドレスを追加 → 「保存」
クライアントID を貼り付け
💡 クライアントIDはこのデバイスのブラウザにのみ保存されます。第三者には送信されません。
